# Data Directory

This directory contains datasets, CSVs, and other data files used by the Chronos-AI project.

## Structure

The data files here are shared between both language versions of the notebooks (English and Portuguese).

## Data Files

Place your data files here:
- GPR simulation data
- LIDAR data
- Training datasets
- Test datasets
- etc.

## Note

Large data files should not be committed to the repository. Consider using:
- `.gitignore` to exclude large files
- Git LFS for versioning large files
- External data storage solutions
